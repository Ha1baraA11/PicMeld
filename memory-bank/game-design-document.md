# 交互与体验设计 (UX Flows)

## 目标场景与用户
- **场景**: 无网与纯离线环境，适用于大规模截图合并与整理。
- **设备**: 小米手机 (Android 8.0/9.0 重点兼容保证)。

## 界面交互流程
1. **启动与选图**: 
   - 单击“选择图片”，调用系统相册兼容 API（允许最多附加当前到 200 张的总图片数）。
   - 选择后，主页显示选中数与预计生成张数。
   - 以3列网格展示图片缩略图，用户可通过右上角 `X` 按钮移出图片。
2. **生成与进度监控**:
   - 触发“开始生成”后，“选择图片”与“开始生成”按钮即锁死（防止重复与重入）。
   - 展现处理进度条与数字进度文本（例如 10/200 张）。
3. **状态完成与异常**:
   - **成功**: 出现简短提示“完成”，随后自动重置应用状态（清空已选列表）。
   - **异常**: 如遇特定图片损坏，终止并以弹窗警告。
